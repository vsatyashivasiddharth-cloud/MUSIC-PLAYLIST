# MUSIC-PLAYLIST
A Java-based Music Playlist Management System using Linked List, Stack, Queue, and Array with a Swing GUI, supporting song management, playback, and file storage.
A fully functional Music Playlist Management System built in Java that demonstrates the practical implementation of data structures and object-oriented programming concepts. The project uses a linked list to store songs dynamically, a stack to maintain recently played songs, a queue for managing the play-next feature, and an array to track top-rated songs.

The application includes a graphical user interface (GUI) using Java Swing, making it easy to interact with the system. It supports key operations such as adding, deleting, searching, displaying songs, and playing audio files. File handling is implemented to ensure that playlist data is saved and loaded efficiently.
